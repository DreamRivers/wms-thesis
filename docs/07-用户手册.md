# 用户手册

## 一、系统简介

WMS 电商仓储物资管理系统,用于管理电商企业的商品、库位、入库、出库、库存与盘点业务。

## 四、系统管理员(admin)操作手册

### 4.1 用户管理
- 进入"系统管理 → 用户管理"
- 可新增/编辑/删除/启停用户、重置密码(默认 123456)

### 4.2 角色管理
- 查看 4 类角色(ADMIN/WAREHOUSE/DEPT_LEADER/EMPLOYEE)
- 角色编码不可修改,关联菜单由管理员维护

### 4.3 操作日志
- 进入"系统管理 → 操作日志"
- 查看所有用户的增删改操作记录,含模块、操作、IP、耗时

## 五、仓库管理员(wh001)操作手册

### 5.1 主数据维护
- **商品管理**:维护商品 SKU,设置安全库存、临期天数
- **仓库管理**:维护仓库信息
- **库位管理**:维护库位(库区/货架/层/列)
- **供应商管理**:维护供应商信息

### 5.2 入库流程
1. 进入"入库管理 → 入库单列表"
2. 点击"新建入库单"
3. 选择供应商、仓库,添加商品明细(选库位、批次、数量、单价)
4. 点击"保存草稿"→ 单号生成,状态 DRAFT
5. 点击"提交"→ 状态变 PENDING
6. 切换为 admin 账号(或在审核页)进行"审核通过"→ 状态 APPROVED
7. 切回 wh001 账号,在列表"执行"按钮 → 录入实收数 → 状态 EXECUTING
8. 点击"完成"→ 状态 FINISHED,库存增加,流水记录

### 5.3 出库执行
1. 员工提交出库申请后
2. 部门负责人(emp001 的部门)审核通过 → 状态 APPROVING
3. 切换 wh001 账号,进入"出库管理 → 出库审批",切换到"待仓管审批"Tab
4. 点击"通过" → 状态 APPROVED
5. 回到"出库单列表" → 点击"发货出库" → 状态 SHIPPED,库存扣减
6. 点击"完成" → 状态 FINISHED

### 5.4 盘点流程
1. 进入"库存 → 盘点管理"
2. 点击"新建盘点单",选择仓库与类型(全盘/抽盘)
3. 系统自动抓取当前库存作为待盘项
4. 点击"详情"录入实盘数,系统自动计算差异
5. 点击"保存实盘数" → 状态 EXECUTING
6. 确认无误后点击"完成并调整" → 状态 FINISHED,差异自动调整库存并写流水

### 5.5 库存预警
- 进入"库存 → 库存预警"
- 系统自动展示低库存/临期商品通知
- 可标为已读或全部已读
- 定时任务每小时自动扫描,无需手动触发

## 六、部门负责人(dept001)操作手册

### 6.1 审批出库申请
- 切换 dept001 账号登录
- 进入"出库管理 → 出库审批"
- 默认 Tab "待部门审批"显示 emp001 提交的本部门申请
- 点击"通过"或"驳回",输入审批意见,提交
- 通过后,状态变为 APPROVING,等待仓管审核

## 七、普通员工(emp001)操作手册

### 7.1 提交出库申请
- 切换 emp001 账号登录
- 进入"出库管理 → 出库申请"
- 选择出库类型(领用)、仓库,添加商品明细
- 输入出库原因,点击"提交申请"
- 状态变 APPLY,等待部门负责人审批

### 7.2 查看个人申请
- 进入"出库管理 → 出库单列表"查看自己提交的所有出库单状态

## 八、常见问题(FAQ)

### Q1:登录提示"密码错误"?
A:请确认账号密码正确,默认密码为 123456。如仍提示错误,可联系管理员重置。

### Q2:菜单只显示一部分?
A:不同角色菜单权限不同。普通员工只能看到"出库管理"等模块。如需更多权限,请联系管理员分配角色。

### Q3:入库单提交后无法审核?
A:请确认当前账号拥有"入库单审核"权限(管理员或仓管均可)。

### Q4:出库审批未通过?
A:出库需经过两级审批:先部门负责人,后仓管。请确认两级审批人都已通过。

### Q5:库存预警没出现?
A:系统每小时扫描一次(整点),如未出现可检查商品安全库存是否设置正确(> 0)。

### Q6:忘记密码?
A:联系系统管理员,在"用户管理"点击"重置密码"按钮,默认重置为 123456。

### Q7:如何导入商品?
A:商品管理页提供模板下载,按模板填写后上传即可。

### Q8:浏览器兼容?
A:推荐 Chrome 100+ / Edge 100+ / Firefox 100+,IE 不支持。
